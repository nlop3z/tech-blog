# Tech-Blog

## Purpose
The purpose of this application is to allow developers to publish their blog posts and comment on other delveopers' posts as well.

## Built with
* JavaScript
* Sequelize
* Heroku
* express-handlebars
* MySQL2
* Express.js
* bcrypt
* dotenv
* connect-session-sequelize
* express-session
